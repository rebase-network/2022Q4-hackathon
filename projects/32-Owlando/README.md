## 作品简介

Evm Helper 是一个帮助EVM开发者通过提问、回答EVM相关开发问题，获得GoerliETH等测试代币的社区互助型DAPP

流程如下：
1、提问者在Evm Helper上提问，放置悬赏的GoerliETH数量，并发布问题。
2、回答者在Evm Helper上回答问题
3、提问者了结问题，提问者选择有帮助的回答者，回答者平分悬赏的GoerliETH
