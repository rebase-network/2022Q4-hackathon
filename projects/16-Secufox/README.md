## 作品简介

SecuFox提供to C及to B的一站式合约安全聚合检测服务，致力于打造web3的安全基础设施及安全服务平台，助力营造安全的Web3生态。
通过SecuFox：
C端用户可以获得针对具体合约的详尽，聚合且全面的合约安全检测。
B端用户可以对自己的合约进行合约开发安全检测，进而规避潜在的安全风险。
